## Limit Order Book data in CNN LSTM model with Early Bird Tickets technique

In high frequency trading, it is common for investment firms to apply neural networks for generating predictions. Several architecture including Convolutional
Neural Network and Long-Short Term Memory to capture spatial anbd temporal information. However, the architecture is usually complicated. By using Early Bird Ticket 
technique, we can prune some paths in the network and only choose thoes paths with weights that lead to strongest performance. In the notebook, it is shown that
this technique can speed up training by 60% while preserving testing accuracy.
